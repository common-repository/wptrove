Collect and display testimonials.

== Description ==

Collect and display testimonials by send a link via email/text/DM's.

**Step # 1** Install plugin.
**Step # 2** Create a page or edit an existing one.
**Step # 3** Add the 'Add Testimonial' block.
**Step # 4** Send the link to someone.
**Step # 5** Visit 'WPtrove' page in admin dashboard.
**Step # 6** Select "Pending" from dropdown and approve/delete the testimonials by your clients.
**Step # 7** Edit/Create the page you want to display the testimonials and add the "Display Testimonials" block.
